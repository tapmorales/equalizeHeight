• equalizeHeight

    It is a plugin Jquery for recalculate de height of elements with float, keeping the layout consize.

  • Check out

    Save this page and try to remove this plugin and check out what happens to each li.

  • Also

    Resize the window and see the magic over the li's

  • How can I use?

    Just set: $('container').children().
    equalizeHeight();

  • And now...

    Every li on the same line will have the same height.

  • Liquid Layout

    If you are using a liquid layout, you can call the method whenever the window is resizeble.

  • Exemple

    $(window).resize(function(){
    $('container').children().
    equalizeHeight();
    })

  • Easy

    Your floats will always became like a table, whith no problem in adjustment Layout.
